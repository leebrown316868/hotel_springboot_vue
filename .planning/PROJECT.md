# 酒店民宿管理系统

## What This Is

一个基于SpringBoot + Vue3 + SQLite的酒店民宿管理系统，作为毕业设计项目。系统支持多角色（管理员、前台、客户），提供完整的酒店业务流程管理，包括房间管理、预订管理、客户管理、数据分析等功能。前端使用Element Plus组件库，后端使用Spring Boot框架，数据库使用轻量级的SQLite。

## Core Value

**能完成答辩的完整酒店管理系统** - 作为毕设项目，系统必须功能完整、运行稳定、演示流畅，能够展示完整的技术栈应用和业务逻辑实现。

## Context

这是一个毕业设计项目，主要目标是展示全栈开发能力。前端已基本完成，需要实现完整的后端API。系统需要支持演示场景，因此需要预置一些示例数据（如房间类型、示例客户等）。项目要求简单实用，避免过度设计。

**前端技术栈：** Vue3 + TypeScript + Element Plus + ECharts + TailwindCSS + Vite
**后端技术栈：** Spring Boot + SQLite + MyBatis/JPA

## Constraints

- **毕设时间**: 需要在答辩前完成 — 优先保证核心功能可用
- **技术栈**: 必须使用SpringBoot + Vue + SQLite — 这是毕设要求
- **外部依赖**: 支付、第三方平台使用模拟数据 — 避免复杂的第三方对接
- **开发原则**: 简单问题简单解决，不要过度复杂化 — 提高开发效率
- **演示需求**: 需要预置示例数据 — 方便答辩演示
